# 本地音乐处理

纯前端的本地音频处理网页应用。所有处理在浏览器内完成，无需上传任何音频数据。

## 功能

- 音频加载 — 拖拽或文件选择器加载 MP3 / WAV / OGG / FLAC / M4A
- 播放控制 — 播放/暂停/停止，进度条点击跳转
- 倍速播放 — 0.25x 至 4x 七档预设，实时调速
- 倒放 — 一键反转音频
- 变调 — +/- 12 半音范围
- 裁剪 — 波形拖拽选择区域，裁剪至选中片段
- A/B 预览 — 原始与处理结果一键切换对比
- 循环播放 — 播放到结尾自动重播
- 波形可视化 — 实时波形绘制，播放位置光标
- 导出 WAV — 将处理结果导出为 WAV 文件下载

## 使用方法

1. 用浏览器打开 index.html
2. 拖拽音频文件到左侧文件区域，或点击选择
3. 使用倍速、倒放、变调等控制调节效果
4. 点击应用处理生成处理版本
5. 通过顶部原始/处理按钮对比
6. 点击导出 WAV 下载处理结果

## 键盘快捷键

Space - 播放/暂停
R - 切换倒放
S - 停止
+/- - 升降调
左/右箭头 - 快退/快进 5 秒
0-8 - 速度预设

## 浏览器兼容

需要在支持 Web Audio API 的现代浏览器中运行。推荐 Chrome 或 Edge（最新版）。
